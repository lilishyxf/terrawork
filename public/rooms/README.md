# 室内场景图 / 瓦片底图(M5 视觉)

按 **zone 命名**放 PNG,前端自动铺进对应区(缺图回退半透明色框,不阻塞):

## 室内图 `public/rooms/<zone>.png`(铺满该区框,会等比拉伸到框尺寸)
按当前布局(town.ts ZONE_POS)的**框尺寸**出图最贴合:

| 文件 | 区 | 建议尺寸(宽×高) | 画什么 |
|---|---|---|---|
| `lobby.png` | 大堂(向导) | 300×140 | 前台/调度台 + 显示器 + "当前目标"看板 |
| `workshop.png` | 工坊小屋 | 290×150 | 工位/电脑/工具(已有参考图) |
| `verify_room.png` | 验证间 | 240×150 | 引爆台/检测设备(爆破工) |
| `review_room.png` | 审查间 | 210×150 | 两审查席 + 砖墙门(候审 NPC 站门口) |
| `yard.png` | 院子 | 840×80 | 草地/栅栏(室外,已有参考图) |

## 地板底纹 `public/tiles/floor.png`(整屋背景,自动平铺)
- 一张**可无缝平铺**的小图(如 64×64 米黄拼砖),前端 tileSprite 平铺满画布。
- 缺图则用纯色底(配置色)。

## 说明
- 这些是**产品资产**(原创/CC0/朋友定制,IP 合规同精灵),入库。
- 框尺寸会随布局微调;改了布局我会更新这里的尺寸表,你再按新尺寸出图。
- `审查间门口` 不再单独成框——候审 NPC 站在审查间门口左侧;`屏幕前(HITL)` 是细条不配室内图。
